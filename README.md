# grunt-mixin-dir

Utility to add directory handling to your grunt task

## Getting Started
Install the module with: `npm install grunt-mixin-dir`

```javascript
var grunt_mixin_dir = require('grunt-mixin-dir');
grunt_mixin_dir.awesome(); // "awesome"
```
